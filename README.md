### **Installation & Usage** üöÄ

Want to run this project locally or fork it for your own portfolio? Follow these steps.

#### **Prerequisites**
-   Node.js (v18 or higher)
-   pnpm package manager
-   Mistral API key (set as `MISTRAL_API_KEY`)

#### **Local Setup**

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/SAIGANESH02/ai-portfolio.git
    cd ai-portfolio
    ```

2.  **Install dependencies:**
    ```sh
    pnpm install
    ```

3.  **Set up your environment variables:**
    Create a `.env.local` file in the root directory:
    ```env
    MISTRAL_API_KEY="your_mistral_api_key_here"
    NODE_ENV=development
    ```

4.  **Run the development server:**
    ```sh
    pnpm dev
    ```

5.  **Open your browser** and navigate to `http://localhost:3000`
